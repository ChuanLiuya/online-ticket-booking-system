# Layout 子路由页面清单（推荐）

1. 首页/正在热映
页面名：Home
用途：展示热映电影、轮播、推荐、快速购票入口

2. 电影列表
页面名：Movies
用途：热映/待映切换、筛选（类型/地区/评分）、搜索入口

3. 电影详情
页面名：MovieDetail
用途：海报/简介/演职员/评分/预告片、选择影院/日期的入口
路由建议：/app/movies/:movieId

4. 影院列表
页面名：Cinemas

5. 影院详情 & 排片（必做）
页面名：CinemaDetail / Showtimes
用途：影院信息、某天排片列表（电影/厅/时间/价格）、进入选座

6. 选座（必做，核心）
页面名：SeatSelect
用途：座位图、不可选/已售、选座规则（最多 N 个、情侣座等）、倒计时锁座
路由建议：/app/showtimes/:showtimeId/seats

7. 确认订单（必做）
页面名：OrderConfirm
用途：场次信息、座位、票价明细、优惠券/活动、提交订单
路由建议：/app/orders/confirm?showtimeId=...

8.支付（必做）
页面名：Payment
用途：选择支付方式、展示应付金额、轮询/回调结果
路由建议：/app/pay/:orderId

9. 支付结果（建议）
页面名：PayResult
用途：成功/失败、查看票/继续购票
路由建议：/app/pay/result/:orderId

10. 我的订单（必做）
页面名：MyOrders
用途：全部/待支付/已完成/已取消，退票（若支持）、订单详情入口
路由建议：/app/orders

11. 订单详情/电子票（必做）
页面名：OrderDetail / Ticket
用途：取票码/二维码、影院/影厅/座位、退款状态
路由建议：/app/orders/:orderId

12. 个人中心（建议）
页面名：Profile
用途：手机号、常用观影人（如需要）、设置
路由建议：/app/me
